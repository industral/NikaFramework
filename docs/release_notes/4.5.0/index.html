• Code structure changed

    • logic now is called scripts . There could be JavaScript files and maybe preprocessor scripts, like CoffeeScript in future
    • dom now is called templates . Here could be not only xhtml files but templates also
    • style now is called styles . Moved to multiplicity
    • img now is called images . Better to have full name
    • impl now is app
    • extra now is assets
    • index.template.xhtml moved to nkf4 folder

  • New build system

    BS was refactored. Dropped functionality creating out folder and saving there files. Decreased build time.
    • YUI Compressor was replaced with Google Closure
    • Added package.json
    • Now it support the following options:
      • --js-source-map - JavaScript Source Maps version 3. Now you can easy debug JavaScript
      • --js-optimization - "simple" or "advanced" JavaScript optimization
      • --xml-optimization - remove extra spaces from XML
      • --sass-debug - now use can use FireSass

  • API changed

    • EDD to load/change page replaced with nkf.core.Controller.load({ init: true });
    • Added new helper for components. How instead of writing: var ns = "nkf.impl.components.page"; var self = $.namespace(ns); extendClass(Login, nkf.core.components.component.PageAbstract); Login.className = "Login"; function Login() { this.className = Login.className; you can use nkf.core.Utils.initClass(nkf.core.components.component.PageAbstract, Login, "Login");
